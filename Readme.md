# 👨‍🌾 Farmer Registration & Authentication System

A full-stack web application built as part of the Cognifyz Level 3 - Task 6 internship program.  
This project implements secure user registration, login, and CRUD operations with database integration and token-based authentication.

---

## 📌 Features

- ✅ User registration with server-side & client-side validation
- ✅ Duplicate phone number check (both frontend + MySQL `UNIQUE` constraint)
- ✅ Secure user login with JWT authentication
- ✅ Password hashing with `bcrypt`
- ✅ Protected API endpoints using token-based authorization
- ✅ Full CRUD support for farmer records
- ✅ Responsive UI using Bootstrap 5
- ✅ EJS templating for server-side rendering
- ✅ Express.js REST API
- ✅ MySQL integration for persistent data storage

---

| Layer           | Technology          |
